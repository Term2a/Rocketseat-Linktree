## Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML e CSS;
- JavaScript;
- Git e Github;
- Figma

## Projeto

O DevLinks é um agregador de links para se usar como cartão de visitas online.
